# ♻️ Auto Back — Плагин для FunPay Cardinal

## 📋 Описание

**Auto Back** — плагин для [FunPay Cardinal](https://github.com/sidor0912/FunPayCardinal), который автоматически выполняет возврат средств по отзывам с заданным количеством звёзд и блокирует покупки от пользователей из чёрного списка. Гибкая настройка действий для каждой оценки, защита автовыдачи, кастомные сообщения и ценовые фильтры.

## 🌟 Возможности

| Функция | Описание |
|---------|----------|
| ♻️ Возврат по оценкам | Автоматический возврат при получении отзыва от 1 до 5 звёзд |
| ⚙️ Индивидуальные настройки | Для каждой оценки свои правила: возврат, сообщение, добавление в ЧС |
| 🚫 Блокировка из ЧС | Мгновенный возврат при покупке пользователем из чёрного списка |
| 🛡 Защита автовыдачи | Если заказ содержит автовыдачу — возврат не выполняется |
| 💬 Кастомные сообщения | Отдельный текст для каждой оценки и для покупателей из ЧС |
| 💰 Ценовые фильтры | Возврат только если сумма заказа в заданном диапазоне |
| ⚡ Мгновенная реакция | Обработка сразу при получении отзыва или нового заказа |
| 🔔 Уведомления | Telegram-уведомления о каждом возврате и добавлении в ЧС |
| 💾 Персистентность | Настройки сохраняются между перезапусками бота |

## 📦 Установка

1. Скачайте файл `auto_back.py` из этого репозитория.
2. Откройте чат с ботом FunPay Cardinal в Telegram.
3. Введите команду: `/upload_plugin`.
4. Отправьте скачанный файл плагина боту.
5. Перезапустите бота.

> **Примечание:** Плагин не требует установки дополнительных зависимостей — используются только стандартные модули FunPay Cardinal.

## ⚙️ Настройка

### Быстрый старт

1. Откройте Telegram-бота FunPay Cardinal.
2. Введите команду: `/auto_back`.
3. Включите авто-возвраты главным переключателем.
4. Перейдите в **⭐ Настройки по оценкам**.
5. Выберите нужную оценку и настройте действия.
6. Готово — плагин начнёт обрабатывать отзывы и заказы из ЧС.

### Пример сценария

**Цель:** При получении 1 звезды — вернуть деньги, добавить в ЧС и отправить сообщение.

1. `/auto_back` → **⭐ Настройки по оценкам** → **⭐️**
2. Включите: **Авто-возврат**, **Добавлять в ЧС**, **Отправлять сообщение**
3. Нажмите **💬 Текст сообщения** и введите текст
4. Настройте ценовой диапазон при необходимости

## 🎛 Управление через Telegram

### Команда

/auto_back


Открывает главное меню настроек плагина.

### Главное меню

| Кнопка | Описание |
|--------|----------|
| 🟢/🔴 Авто-возвраты | Главный переключатель плагина |
| 🟢/🔴 Сообщение из ЧС | Отправлять сообщение при покупке из ЧС |
| 🟢/🔴 Возврат из ЧС | Возвращать деньги при покупке из ЧС |
| 💬 Текст сообщения ЧС | Редактировать текст для покупателей из ЧС |
| ⭐ Настройки по оценкам | Перейти к настройке действий по звёздам |
| Мин/Макс цена | Ценовой диапазон для возвратов из ЧС |

### Настройки по оценкам

Для каждой оценки от 1 до 5 звёзд доступны:

| Параметр | Описание |
|----------|----------|
| Авто-возврат | Выполнять возврат при этой оценке |
| Добавлять в ЧС | Добавить покупателя в чёрный список |
| Отправлять сообщение | Отправить кастомный текст покупателю |
| Текст сообщения | Индивидуальный текст для этой оценки |
| Мин/Макс цена | Ценовой диапазон для срабатывания |

## 🔧 Технические детали

- **Хранение данных:** Настройки сохраняются в:

storage/plugins/auto_back/settings.json


- **Потокобезопасность:** Сохранение настроек защищено `threading.Lock`.

- **Атомарная запись:** Используется временный файл + `os.replace` для защиты от повреждения данных.

- **Защита автовыдачи:** Перед возвратом из ЧС проверяется наличие `order_secrets` — если товар уже выдан, возврат не выполняется.

- **Обработка событий:** Плагин подписан на `BIND_TO_NEW_MESSAGE` (отзывы) и `BIND_TO_NEW_ORDER` (заказы из ЧС).

- **Проверка статуса:** Перед возвратом проверяется, не был ли заказ уже возвращён.
